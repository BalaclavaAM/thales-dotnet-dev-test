Welcome to the results of the .NET Developer Testing:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

This project was developed using .NET Core 7 and Angular 15, utilizing IIS Express for local development and testing. The Visual Studio SDK 2022 was used for unit tests and other development tasks.

It's worth noting that sometimes the API may respond with a 429 error, indicating that the server is experiencing too many requests (Rate Limit Exceeded). This can lead to a temporary inability to show data. In such cases, it's recommended to implement proper error handling and possibly implement rate limiting strategies to prevent overwhelming the API server.
